00 · Architecture — why this is simpler to wield

Tier 1 · Primitive

One brand hue in OKLCH. --gold-500, the neutral ramp, signal hues. Raw, perceptual.

Tier 2 · Semantic

Roles, derived via color-mix(): --accent, --bg, --surface-1, --text-2, --line-gold, --on-accent, state & focus.

Tier 3 · Component

The .bx-* primitives consume only semantic tokens — so a re-skin never touches a component.

All inside @layer boom.tokens, boom.base, boom.components, boom.utilities → low cascade priority → a page's own CSS always wins. Safe to import anywhere, today.
